# Markdown Rehberi

Markdown, düz metin kullanarak biçimlendirilmiş belgeler oluşturmanızı sağlayan basit bir işaretleme dilidir.

## Başlıklar

```
# Başlık 1
## Başlık 2
### Başlık 3
#### Başlık 4
```

## Metin Biçimlendirme

| Sözdizimi | Sonuç |
|-----------|-------|
| `**kalın**` | **kalın** |
| `*italik*` | *italik* |
| `***kalın italik***` | ***kalın italik*** |
| `~~üstü çizili~~` | ~~üstü çizili~~ |

## Listeler

**Sırasız liste:**
- Elma
- Armut
- Kiraz

**Sıralı liste:**
1. Birinci adım
2. İkinci adım
3. Üçüncü adım

## Bağlantılar ve Resimler

```
[Bağlantı metni](https://ornek.com)
![Resim açıklaması](resim.png)
```

## Kod

Satır içi kod için tek backtick: `kod buraya`

Kod bloğu için üç backtick:

```python
def merhaba():
    print("Merhaba, Dünya!")
```
